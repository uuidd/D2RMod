 这是一个伪转换器。 它加载自定义（修改）文件以用作 D2RMM 脚本的基础文件。

这个模组必须在模组中排名第一才能工作。

主要有两个应用：

1. 将 非 D2RMM 模组与其他 D2RMM 模组合并（即插即用，无需编写代码）：

-加载非D2RMM mod（必须为当前版本的D2R配置此mod）：
将非 D2RMM mod 的数据文件夹的内容复制到“MyMod”文件夹（在 FileLoad mod 文件夹中找到）。
MyMod 文件夹应包含诸如“global”、“hd”等文件夹 。

- 将 FileLoad 设置为 mod 顺序中的 #1。
-继续加载 D2RMM 模组。

使用 D2RMM 午餐游戏，或者如果您的 mod 使用数据文件夹之外的文件：用 D2RMM.mpq/data 目录替换（覆盖）'YourModName'/data 的内容并午餐您的 mod（AFAIK 没有使用数据文件夹之外的文件的 D2RMM mod ).

如果您在基本模组中遇到错误或功能中断，则意味着两者之间存在冲突。

注意 - 这是“使用风险自负”，因为不会检测到某些冲突。 实验并玩得开心！

2. 文件生成/操作 （ 需要编写 Java Script）： （我用它来对我的个人 mod 进行繁琐的更改）

- 将您的 mods txt 文件加载到 LoadFile。
-创建一个新的虚拟模组来操作 txt/json 文件，示例：
•为您创建的项目添加大量描述（JSONEdit 中的示例），
•“全部替换”功能（例如：Skill Synergies 在奖励后使用技能级别（而不是基础级别）-> 将 skills.txt 和 skilldesc.txt - TXTEdit 中所有出现的 blvl 替换为 lvl），
•调整某些属性的所有单元格值
（例如：将垃圾掉落几率降低 50%，但将差异添加到 treasureclassex.txt 中预定义的“空”宝藏类中，这将消除混乱但不会影响掉落几率）

- 在 D2RMM 中点击 Install Mods，这将在游戏目录的 D2RMM.mpq 文件夹中创建新文件，只需将文件移动到您的 mod 文件夹中即可应用您所做的更改。

这将为您节省多少时间取决于编辑，我大多数时候更喜欢这样做，因为一个一个的单元格编辑非常无聊，您可能会错过一两个条目，这种方法不会错过任何东西。

注意：D2R txt 文件中的引用通常不一致，因此请检查您的编辑是否可以自动进行。 即 Skill Synergy 奖励值至少有 3 个不同的参考主题，并且对它们的描述不一致，因此很难在全局范围内进行编辑。
